GitHub命令集锦

Java24K 2022-06-01

# 一.常用Git命令

# Git创建仓库

创建Git仓库即在指定目录下生成一个 .git 目录，该目录包含了资源的所有元数据。

  • 当前目录创建仓库。
    git init
  • 指定目录创建仓库。
    git init myrepo

# 将文件添加到暂存区

  • 添加当前目录下的所有文件到暂存区
    git add .
  • 添加指定目录到暂存区
    git add [dir]
  • 添加一个或多个文件到暂存区
    git add [file1] [file2] ...

# 设置提交代码用户信息

  • 设置本地项目库配置
    git config user.name "java24k"
    git config user.email "772835869@qq.com"
  • 设置全局项目库配置
    git config --global user.name "java24k"
    git config --global user.email "772835869@qq.com"

# 提交暂存区到本地仓库

  • 提交暂存区到本地仓库中
    git commit -m "xxxx"
  • 提交暂存区的指定文件到仓库中
    git commit [file1] [file2] ... -m "xxxx"

# 创建分支命令

   git branch (branchname)

# 重命名分支命令

   git branch (-m | -M) <oldbranch> <newbranch>

-M代表强制重命名，即使新分支已经存在

  • git 中一些选项解释

  1. -d --delete：删除

  2. -D --delete --force的快捷键

  3. -f --force：强制

  4. -m --move：移动或重命名

  5. -M --move --force的快捷键

  6. -r --remote：远程

  7. -a --all：所有

# 添加远程仓库并指定简写

  • 添加远程仓库
    git remote add [shortname] [url]

# 推送内容到远程仓库

  • 推送到远程仓库
    git push [-u] [remote-name] [branch-name]

如果当前分支与多个主机存在追踪关系，那么这个时候-u选项用于指定默认主机，这样后面就可以不加任何参数使用git push。
